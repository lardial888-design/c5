# Калькулятор

Браузерный калькулятор с модульными и E2E-тестами.

## Функциональность

- Базовые математические операции (сложение, вычитание, умножение, деление)
- Поддержка десятичных чисел
- Обработка ошибок (деление на ноль)
- Поддержка клавиатурного ввода
- Очистка и удаление символов

## Тестирование

### Модульные тесты (Vitest)

Запуск модульных тестов:

```powershell
# Запуск в watch режиме
npm test

# Запуск один раз
npm run test:run

# Запуск с покрытием
npm run test:coverage
```

### E2E тесты (Playwright)

Запуск E2E-тестов:

```powershell
# Запуск всех E2E-тестов
npm run test:e2e

# Запуск с UI
npm run test:e2e:ui

# Запуск в headed режиме (с видимым браузером)
npm run test:e2e:headed
```

E2E-тесты проверяют:
- Ввод чисел через кнопки и клавиатуру
- Все математические операции
- Деление на ноль
- Очистку и удаление символов
- Отображение результатов
- Интеграционные сценарии

## Установка

```powershell
npm install
```

## Использование

Откройте `calculator.html` в браузере или используйте локальный сервер:

```powershell
npx http-server . -p 3000
```

Затем откройте http://localhost:3000/calculator.html
